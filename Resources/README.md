# 图标资源说明

本文件夹用于存放靶子图标资源文件。

## 需要的图标文件

请将以下 PNG 格式的图标文件放置在此文件夹中：

- `recycle_bin.png` - 回收站图标（48x48 像素）
- `new_folder.png` - 新建文件夹图标（48x48 像素）
- `excel.png` - Excel 文档图标（48x48 像素）
- `word.png` - Word 文档图标（48x48 像素）
- `text_document.png` - 文本文档图标（48x48 像素）

## 图标要求

- 格式：PNG（支持透明背景）
- 尺寸：48x48 像素（或系统标准图标大小）
- 建议：使用与 Windows 桌面图标风格相似的图标

## 占位符

如果图标文件不存在，程序会自动生成彩色方块作为占位符：
- 回收站：红色
- 新建文件夹：蓝色
- Excel：绿色
- Word：深蓝色
- 文本文档：橙色

## 获取图标

您可以从以下来源获取图标：
1. Windows 系统图标提取工具（Windows 10/11）
2. 在线图标资源网站
3. 自行设计制作

> 💡 **提示**：程序会自动通过 Windows API 获取系统图标，无需手动添加资源文件。如需自定义图标，可将 PNG 文件放入此文件夹。
